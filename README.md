# IAEditais

O IAEditais é um projeto para a fundação Oswaldo Cruz (FioCruz), gerenciado pelo grupo de pesquisa NPAI (Núcleo de Pesquisa Aplicada e Inovação).

## Objetivo

A plataforma dará suporte à análise de editais de compras públicas. A IA generativa irá interpretar os editais de acordo com documentos oficiais fornecidos, poupando o tempo de revisores e agilizando o processo.

> Framework utilizado: **_Next.JS_**
